# ScoreSystem

### 软件介绍：
该软件对正方教务管理系统中的成绩进行爬取，并做了相对的整理，结合了前端GUI进行数据显示。

### 环境介绍：
- python == 3.6.3
- beautifulsoup4 == 4.6.0
- PyQt5 == 5.11.2
- requests == 2.19.1
- pyinstaller == 3.4.dev0+7817d3fac

### 使用介绍：
配置好环境，直接运行main.py文件即可。
如果是其他学校的正方教务系统，可以根据自己学校情况修改education_system.py文件中的url测试。
